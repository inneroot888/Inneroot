Inneroot V6.9.1 — 安全的新一日重置流程

今次修正：
- 日期轉換後，昨日心情、日記、最觸動問題及給自己的話不會繼續出現在今日
- 清空前會先將昨日內容保存到 innerootInsightHistory
- 新一日首次開啟會顯示「新的一天開始了」
- 第一次安裝新版而日期來源不明時，可選擇：
  1. 開始今天：保存舊內容後清空
  2. 保留目前內容：不作清空
- 同一日重新開啟網站，今日未完成內容仍會保留
- 今日抽牌功能維持每日一張

沒有修改：
- Logo、背景、插畫
- 綠色按鍵及心情圖案
- 功能卡及底部功能列
- 深入探索及內在洞察內容
- CSS 與桌面版版面
