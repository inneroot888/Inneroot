# Inneroot 視覺互動原型

這個版本以已確認的首頁設計作為完整視覺，並在上面加入真正可按的互動區域。

已有功能：
- 今日心情
- 隨機抽牌
- 潛意識日記
- 複製 AI 探索指令
- 潛意識洞察說明
- 成長足跡說明
- 桌面及手機顯示

## GitHub 手機上傳
1. 下載 ZIP 並在「檔案」App 解壓。
2. GitHub repository 內按 `uploading an existing file`。
3. 上傳 `app`、`public`、`package.json`、`README.md`。
4. 按 `Commit changes`。
5. 到 Vercel Import 這個 repository，再按 Deploy。
